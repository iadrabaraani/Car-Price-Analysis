# Project Overview
This project performs Exploratory Data Analysis (EDA) on a used car listings dataset using Python and Pandas.  
The objective is to clean the data, answer business-related questions, and visualize key trends in car pricing.

---

# Objectives
- Load and inspect real-world automobile data
- Perform data cleaning and quality checks
- Answer analytical queries using Pandas
- Create meaningful visualizations
- Extract business insights from the data

---

# Dataset
- File: `car_prices.csv`
- Description:
  Contains used car listings with attributes such as selling price, make, model, year, condition, odometer reading, color, interior, and state.

---

# Tools & Libraries Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

# Key Tasks Performed
- Data ingestion and inspection
- Handling missing values and duplicate records
- Aggregation and filtering using Pandas
- Feature engineering (Car Age calculation)
- Correlation analysis
- Data visualization with business insights

---

# Visualizations Included
- Correlation heatmap of numerical features
- Average selling price by year
- Selling price vs odometer trend
- Number of cars sold per state
- Average price by condition ranges
- Distribution of selling prices by car color

---

# How to Run the Project
1. Ensure `car_prices.csv` and `Car_Price_Analysis.ipynb` are in the same folder
2. Open the notebook in Jupyter Notebook
3. Run all cells from top to bottom
